# part 1

vuejs@3.0.1 설치 (node js 우선 설치)
>> npm install vuejs

cli 설치
>> npm install -g @vue/cli

기본 프로젝트 생성 (ver. 3)
>> vue create app

명령어 입력 (서버 실행)
>> cd app
>> npm run serve

localhost 접속
-> 환경 설정 완료
